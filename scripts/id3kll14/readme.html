ID3KILL for Windows v1.4 Documentation
[ released 26.12.2001 ]

ID3KILL removes ID3v1 and ID3v2 tags from MP3 files.

Overview:

This freeware tool removes and lists ID3v1 & ID3v2 tags from MP3 files.

ID3 is a general tagging format for audio, which makes it possible to store meta data (album, year, genre, ...) about the song inside the audio file itself.

This is useful if you encounter incorrect tags in your MP3 files or if you wish to upgrade the tags in your files. ID3KILL has the option of only removing ID3v1 or ID3v2 tags should you wish to preserve one of the two, and can recursively search within subdirectories if you want total annihilation. ID3KILL can even modify files marked 'read-only', provided the storage medium allows it. Simple, easy, removes ID3 tags within a few clicks!

The latest adds an option to clean up M3U and PLS playlists, removing title and length information (EXTINF) - this saves another few kbytes of hard disk space. For more details see included HTML documentation.

View some screenshots!

 
Features:

  • no file limit
  • very fast processing
  • list-only abilities
  • option to preserve ID3v1 or ID3v2 tags
  • compatible with upcoming ID3 tag standards (see id3.org)

 
Options:

  • Include Subdirectories: self-explaining

  • Remove ID3v1 Tags: this setting enables ID3v1 removal from MP3 files
  • Remove ID3v2 Tags: enables ID3v2 removal from MP3 files

  • Clean M3U Lists: removes any lines starting with # from M3U playlists. This includes the popular EXTINF information, eg. used by WinAMP.
  • Clean PLS Lists: removes 'Title' and 'Length' information from PLS playlists.

  • Ignore attributes: if enabled, ID3KILL even modified files marked as 'read-only' or 'system'. Attributes will stay intact!
  • List only modified files: if unchecked, all files found will become listed instead of only modified files.

  • Clean Registry: remove all ID3KILL settings from the Windows registry. You can use this to reset all options or before uninstalling ID3KILL.

 
Command line options:

id3kill.exe directory /s /t1 /t2 /i /r /c (see README.TXT for details)
You can also drag and drop a directory on the ID3KILL executable.

 
Shortcut Keys:

  • ALT-S: include subdirectories
  • ALT-1: remove ID3v1 tags
  • ALT-2: remove ID3v2 tags
  • ALT-M: clean M3U lists
  • ALT-P: clean PLS lists
  • ALT-I: ignore file attributes
  • ALT-L: list only modified files
  • ALT-P: popup message when finished

  • ALT-G: start removal (go)
  • ALT-H: open HTML help
  • ALT-Q: quit ID3KILL
  • ALT-C: clean windows registry
